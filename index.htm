Week 5 (7/28-8/3)

問題與進度 : 因為Quartus II 18.1版本與13.1版的設定方式不一樣，所以下載 Quartus II 13.1練習影片上issp的設定與用法
下載Quartus II 13.1之後，RTL Simulation無法跑出波形圖，但資料夾路徑確實有NativeLink.exe的檔案，想找Simulation Mode更改卻找不到Simulation Mode
RTL Simulation顯示不出波形的問題已經解決了，但因為 13.1 ROM的原設定是VHDL file，所以沒辦法初始化 ROM

如何解決遇到的問題 : 這是檔案路徑的問題，在跟其他組員討論的時候請其他組員測試發現都有這個路徑的問題
之後詢問 Ai， Ai表示將資料路徑更改下載位置後就成功開啟 RTL Simulation ， ROM 初始不了的原因如上述(因版本設定問題)，
只要將原設定的 VHDL file 改成 Verilog HDL file 就能成功初始化 ROM 跑出波形圖
